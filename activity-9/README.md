This activity consisted of creating a web application using Laravel that includes a basic authentication system. It was configured so that guest users can access a public landing page and registration and login pages, while registered users can log in and access a protected dashboard. 

The navigation bar dynamically changes depending on whether the user is authenticated, showing login and registration options for guests and dashboard access for logged-in users. Additionally, access to the dashboard is restricted using authentication middleware to ensure that only registered users can view it, also, a CSS file was implemented to provide a more AI look.
